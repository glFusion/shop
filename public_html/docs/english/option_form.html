Attribute settings

Product Name/SKU
Select the product to which this option will apply. The product must exist before you can apply option to it.
Attribute Name
This is the name of the option : Size, Color, Capacity, etc. See "Attribute Groups" to set up the group names.
Attribute Value
The "Attribute Value" is the display value of the option. For example, if the Attribute Name is "Color", an Attribute Value might be "Blue." If there is a price for the option you might want to include it, e.g. "Blue (+2.00).""
Attribute Price
Enter the incremental price for this specific option. This is the extra charge that will be applied to the product if this option is selected.

This value may be negative if a discount is given for this option. The price is not displayed.

Position After
Select the option after which this option should appear in the selection list.
SKU
Enter an optional SKU code for this option value. This should be a short, consistent indicator such as "BK" for "Black" or "S" for "Small". If present, this value will be included when the overall product SKU is created.
Enabled?
Check this box if this option value is enabled. You might want to disable a value if it is temporarily out of stock, for example.

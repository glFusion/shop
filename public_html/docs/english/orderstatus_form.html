Shop Order Status Form

Name
The short descriptive text for the status, e.g. "processing" etc.
Enabled?
Indicate that the order status can be manually set on an order. Does not affect automatic status changes such as when payment is received.
Notify the Buyer?
Indicate whether an email is automatically sent to the buyer when the status is changed to this value.
Notify the Administrator?
Indicate whether an email is automatically sent to the shop administrator when the status is changed to this value.
Order is Valid?
Indicate whether reaching this status indicates that the order has been accepted and validated.
Eligible for Affiliate Payout?
Indicate whether orders with this status are eligible for affiliate payments. Typically completed orders are eligible.

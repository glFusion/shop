Shop Option Groups

Option Name
This is the name of the option group: Size, Color, Capacity, etc. Names are global but the actual option values will be assigned per-product.
Option Type
Select the type of option display. Selection Lists and Checkboxes are currently supported.
Position After
Select the existing option group after which this one should be displayed. This affects the appearrance on the product detail form and also how SKUs are created.

Shipping Methods

Carrier
Select a predefined carrier code for this shipper. This is used to gather carrier-specific information such as tracking URLs. Select "Other" if your carrier is not in the list.
Name
Enter a name for this shipping method, e.g. "Flat Rate Ground". This is required.
Enabled
Select whether this shipping method is enabled and can be selected during checkout
Requires Shipping Address?
Check this box if a shipping address is required for physical or taxable items. Some methods such as local pickup may not require an address.
Sales Tax Location
Select whether sales tax is charged based on the destination or the origin. Items being picked up at the store are normally taxed at the origin rate.
Include Item Fixed Shipping Cost?
Check if this shipping method should add the per-item fixed shipping cost to the total calculated amount. If unchecked, then the shipping amount calculated will be the only amount charged.
Min. Shipping Units
Indicate the minimum shipping "units" to be handled by this method. Zero is converted to .0001 because zero indicates free or no shipping charge.
Max. Shipping Units
Indicate the maximum shipping "units" that can be handled by this method. Enter zero for unlimited.
Free Shipping Threshold
Check the box to enable free shipping for this method, and enter the order value needed to qualify for free shipping.
Rate Quote Method
Select whether shipping costs will be calculated simply by a unit-based rate table entered below, or if you will configure packages and services to use the shipper's quote API. If "Package Rates" is selected the rate table on this page is ignored.
Rate Table
Enter a short description, maximum allowed units, and per-parcel rate for each type of package handled by this method. Units may be fractional. Click the icon to add new rates. Not used unless "Rate Table" is selected above.
Authorized Group
Select the glFusion group that is allowed to use this shipping method. The default is All users.
Starting Date
Enter or select the starting date when this shipping method will be available.
Ending Date
Enter or select the ending date after which this shipping method will not be available.

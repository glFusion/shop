Package Definition

Description
A short description of the package.
Package Width
Enter the width of the package, using the configured unit of measure.
Package Height
Enter the height of the package, using the configured unit of measure.
Package Length
Enter the length of the package, using the configured unit of measure.
Package Max Weight
Enter the maximum allowed weight to be shipped via this package.
Unit Capacity
The maximum number of shipping units that can be packed into this package.
Carrier Options
Select the package and service type for this package when shipped by various carriers. For the Service Code, select "Not Available" if this package cannot be shipped by the carrier, or "Fixed Rate" and enter a fixed amount as the rate to override the rate quote API.
